# OpenDray Project Goal Template

Build and operate **[PRODUCT NAME]**, a **[SHORT PRODUCT DESCRIPTION]**.

The system must remain modular, secure, testable, observable and easy to change. The authoritative engineering rules are stored in `AGENTS.md`; architecture decisions are stored under `docs/architecture/decisions/`.

All agents must read those files before substantial implementation or architectural changes. Repository policy overrides remembered context and OpenDray memory when they conflict.
